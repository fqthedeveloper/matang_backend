Place a Devanagari TrueType font file here so the PDF can render Marathi text correctly.

Expected filename:
NotoSansDevanagari-Regular.ttf

Download from Google Noto fonts or another Devanagari font provider, then place the font file in this folder.

If the font file is not present, the PDF will still generate using Helvetica, but Marathi text may not render properly.
